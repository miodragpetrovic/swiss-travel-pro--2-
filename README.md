# Switzerland Travel Pro (Demo)

Modern, professional travel website starter inspired by Switzerland Travel Centre — built with **React + Vite + TailwindCSS**.

## Features
- DE/EN language switch with route prefix (`/de`, `/en`)
- Search hero (Wann/Was), results page with sorting, filters, pagination
- Reisen + Gruppenreisen listing
- Detail page with itinerary, inclusions, request modal
- Favorites (heart) persisted in `localStorage`
- Contact page + FAQ + About + legal pages (Impressum/Datenschutz/AGB)

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Replace images
All trip images use public URLs (Unsplash). Replace in `src/data/trips.ts`.
